# Store Assets

`screenshots/` contains unframed screenshots captured with mock data for every supported platform:

- `basalt`
- `diorite`
- `emery`
- `gabbro`

Each platform has the same four submission views:

- chat list
- messages
- photo message
- action list

Upload these through the Pebble/Rebble developer portal as listing screenshots. The watch launcher icon is not stored here; it is `resources/images/menu_icon.png` and is bundled as the `menuIcon` resource in `package.json`.
