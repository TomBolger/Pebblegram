Login setup

  1. Open my.telegram.org and sign in with your Telegram phone number.
  2. Choose API development tools. If Telegram asks you to create an app, fill in the required fields and submit it.
  3. Copy the App api_id into Telegram API ID, then copy the App api_hash into Telegram API hash.
  4. Enter your phone number in international format, including the plus sign and country code, such as +15551234567.
  5. Tap Save. Telegram should send a login code in the Telegram app or by SMS.
  6. Open settings again, enter the Telegram login code. If you use Telegram two-step verification, enter your Telegram cloud password too.
  7. Tap Save again to finish login. If Telegram still asks for a password, return here one more time and enter it.

Can't log in? Need help?

Leave feedback or report login issues on the Pebblegram GitHub issues page. Include your Pebble model, phone OS, what step failed, and the exact message shown on the watch if you can read it.
